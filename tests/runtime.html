Numbas test checklist

  1. Run the JME tests
  2. Compile and run tests/stability.exam.
  3. Run it in Chrome, Firefox and IE8.
  4. Upload to Moodle, submit some answers, exit and re-enter.

Things to look for

  • Front page looks OK
  • Question selector looks OK - marks shown, no feedback.
  • Total is correct.
  • Write an incorrect answer and submit part. Look at feedback.
  • Correct the answer and resubmit. Look at feedback.
  • Leave an input blank and submit. Should give a warning.
  • Write nonsense in the JME box. Should give a helpful error message.
  • Check that shuffled MCQs are shuffled
  • Submit all parts.
  • Move to the next question, then move back.
  • Press pause. Check pause screen looks OK. Press resume.
  • Work through all the questions.
  • Use the "try another question like this" button.
  • Check that the correct answer matches the new question.
  • Press reveal. Check the answers are all filled in and correct.
  • The advice should be shown.
  • Make a note of scores and end the exam.
  • Check all the info looks ok.
  • Check no. of questions attempted.
  • Press Exit.
